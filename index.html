Gitz

The Vibable Programming Language

Our Story

Gitz was born from the desire to make coding as natural as conversation. We believe that programming should have personality, and that code can be as expressive and fun as the people who write it.

Built with a unique blend of broken English and rigorous static checks, Gitz lets you write code that reads like dialogue while ensuring that variables are declared, types match, and errors are caught before runtime.

Whether you're a seasoned developer or just starting out, Gitz offers an approachable, vibable coding experience where creativity meets robust engineering.

Example Programs

Variables and Printing

Make name: text = "Alice";
say("Hello", name);

Loops

Make x: num = 0;
Keep x smaller 10 {
    say(x);
    x = x plus 1;
}

If-Else Statements

When x bigger 5 {
    say("Big number!");
} orElse {
    say("Small number!");
}

Functions

Show greet(name: text) -> text {
    give "Hello " plus name;
}

Lists

Make fruits: list = ["apple", "banana", "cherry"];
say(fruits[0]);
